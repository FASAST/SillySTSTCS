Star Trek: Starship Tactical Combat Simulator
Computer Conversion by Jason Robinson

 Star Trek: Starship Tactical Combat Simulator was a board
game published by FASA in the Eighties. The game was
designed as a supplement to FASA's Star Trek Role-Playing
Game, but could be played as a standalone game.

I enjoyed playing ST:STCS in its heyday, and in 1991 I
decided to create a computer version of the game as a
programming exercise. I started off using AMOS on an
Amiga , ported the game to DOS using Borland C in 1994,
and around 1996 I ported the game to Windows using
Visual Basic. I still work on the program when the mood
strikes me.

The computer version attempts to faithfully recreate the
game with the addition of computer-controlled opponents.
It has modern, Star Trek -inspired interfaces, graphics and
sounds. The game includes tools to create custom ships
and scenarios.

 Star Trek: Starship Tactical Combat Simulator v2.05

 A computer conversion of an FASA's board game, Star
Trek: Starship Tactical Combat Simulator. It's a 2D turn-
based strategy game in which you take control of a starship
and proceed to attempt to destroy other starships before
they destroy you. The game can be played by multiple
humans (in "hot seat" mode) or against computer-
controlled opponents. Editors are integrated into the game
so users can customize the provided ships and scenarios or
add completely new ones. Additional sounds and bitmaps
can also be added. A typical game lasts from 5-10 minutes
for a one-on-one scenario, to about an hour or more for
scenarios with over 10 ships.

The game requires Windows 95/98/2000/XP.

Windows 7 users need to download and install the Visual
Basic 6 runtime files from Microsoft.

Star Trek: Starship Tactical Combat Simulator Alternate
Rendered Graphics Add-On

The Alternate Rendered Graphics Add-On collection is a set
of new bitmaps, ship files, and scenarios for the Star Trek:
Starship Tactical Combat Simulator program. The intent of
this package is to give the game a more modern look, as
opposed to the original's adherence to the board game art.

The bitmaps were generated by rendering 3D models
originally created for Interplay's Starfleet Command
computer game and its sequels (credits to the model
makers are included in the zip). All models were consistently
lit and then rendered roughly to the scale of one pixel per
three meters. The smallest ships are rendered slightly out of
scale to keep the resulting image recognizable.

The set includes every ship class from in the standard FASA
rulebook along with all FASA-produced variations (from the
ship recognition manuals). Additional rendered versions of
map elements are also included to keep scenario maps
visually consistent with the new models.

The new scenarios merely overwrite the default ones with
versions that utilize the newly rendered map elements.

Note that these files are intended to overwrite all of the
default ship and scenario files. If you want to keep the old
ships and scenarios, make sure to back them up or rename
them (otherwise, you'll need to re-install the game to get
them back). No user-made content is affected by this
package unless such content uses matching file names.
